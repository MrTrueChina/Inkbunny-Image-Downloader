Inkbunny Image Downloader Option

Default (single)
[${authorName}] ${imageName}_Inkbunny_${imageId}

Default (multi)
[${authorName}] ${imageName}_Inkbunny_${imageId}_${page}

Element Variables
Author Name ${authorName}
Author ID ${authorId}
Image Name (Title) ${imageName}
Image ID ${imageId}
Page ${page}
